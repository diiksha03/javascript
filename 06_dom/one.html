--> JavaScript DOM Properties & Methods (Quick Notes) 1. innerHTML ✅ Returns: All HTML content (including tags) of the selected element. 🛠️ Use when: You want to get or set the HTML structure. 📌 Example: document.getElementById("demo").innerHTML = "Hello"; 2. textContent ✅ Returns: All the text content, including hidden elements. 🛠️ Use when: You need raw text without HTML tags. 📌 Example: document.getElementById("demo").textContent; 3. innerText ✅ Returns: Only the visible text of the node. 🛠️ Use when: You want text as displayed (excludes hidden content). 📌 Example: js Copy code document.getElementById("demo").innerText; 🔍 DOM Selectors 4. querySelector() ✅ Selects: The first matching element using a CSS selector. 📌 Example: document.querySelector(".my-class"); 5. querySelectorAll() ✅ Selects: All matching elements as a NodeList (like an array). 📌 Example: js document.querySelectorAll("p"); ⚙️ Attributes 6. getAttribute() ✅ Purpose: Gets the value of an attribute on an element. 📌 Example: let link = document.getElementById("myLink").getAttribute("href"); 7. setAttribute() ✅ Purpose: Sets/updates an attribute and its value. 📌 Example: document.getElementById("myLink").setAttribute("href", "https://google.com");
